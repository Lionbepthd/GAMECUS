Game Petualangan Lionbepthd

Kumpulan mini game seru dalam satu dunia interaktif!

Deskripsi

Lionbepthd adalah game buatan anak bangsa yang menyatukan beberapa permainan klasik dalam satu aplikasi ringan dan seru. Cocok dimainkan untuk semua umur langsung dari browser atau sebagai aplikasi Android!

Fitur Unggulan

  • Game Ular Tangga dengan soal kuis sejarah
  • Pac-Man klasik dengan peta pilihan
  • Game catur dengan skor tersimpan
  • Tampilan sederhana dan kontrol mudah
  • Tidak perlu koneksi internet setelah diunduh
